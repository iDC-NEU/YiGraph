## 一、项目规划

### 1.1 项目概述

YiGraph（易图）是一套基于 **AAG（Analytics-Augmented Generation，分析增强生成）** 框架的端到端图数据分析智能体系统。用户通过自然语言描述业务问题，系统即可自动抽取实体与关系、构建结构化图数据、规划分析流程、执行图算法计算，并生成清晰、可解释、可追溯的分析报告。

### 1.2 核心技术架构

| 层次 | 组件 | 技术选型 |
|------|------|----------|
| 推理引擎 | 大语言模型 | OpenAI API / Ollama（本地） |
| 图计算引擎 | 图算法执行 | NetworkX + Neo4j |
| 检索引擎 | 向量+图检索 | Milvus + BGE Embedding |
| 任务规划 | RAG 增强推理 | LlamaIndex |
| Web 前端 | 交互界面 | Flask + Flask-SocketIO |

### 1.3 功能规划

#### 已完成（v1.1.0）

- [x] 完整的图计算引擎（基于 NetworkX 和 Neo4j）
- [x] 智能任务规划与执行（知识驱动 + 算法核心）
- [x] 200+ 种图算法支持，覆盖 21 大类别
- [x] 多数据源支持（图数据 / 文本数据）
- [x] 三种运行模式（普通 / 交互 / 专家）
- [x] Web 交互界面与终端交互模式
- [x] 中英文双语支持

#### 规划中（v2.0.0）

- [ ] 交互体验更强的用户页面
- [ ] 算法知识库的动态扩展与持续更新

### 1.4 适用场景

- 金融反洗钱与可疑交易分析
- 电商风控与羊毛党识别
- 企业关联与风险排查
- 园区/城市事件分析
- 供应链风险分析

### 1.5 关键依赖

- Python >= 3.11
- Neo4j 3.5.25（图数据库）
- Milvus（向量数据库，可选）
- Conda 虚拟环境（推荐）
